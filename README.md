#  Network Device Scanner

##  Описание
**Network Device Scanner** - это многопоточная утилита на Python для быстрого обнаружения активных устройств в локальной Wi-Fi или LAN сети. 

Инструмент автоматически определяет подсеть пользователя и производит Ping Sweep (ICMP-сканирование) всех 254 возможных адресов. Благодаря использованию пула потоков (`ThreadPoolExecutor`), сканирование выполняется параллельно и занимает считанные секунды.

##  Возможности
* **Автоматическое определение подсети:** Скрипт сам находит ваш локальный IP-адрес через `socket` и вычисляет диапазон для сканирования.
* **Многопоточность:** Использование 100 одновременных потоков (`concurrent.futures`) ускоряет процесс в сотни раз по сравнению с последовательным сканированием.
* **Кроссплатформенность:** Модуль `platform` автоматически подбирает нужные синтаксические ключи для команды `ping` в зависимости от вашей ОС (Windows, Linux или macOS).
* **Чистый вывод:** Модуль `subprocess` перехватывает системный вывод, отображая только реально найденные и работающие IP-адреса.

##  Установка и запуск
1. Убедитесь, что у вас установлен **Python 3.x**. Дополнительные библиотеки скачивать не нужно — используются только встроенные модули.
2. Склонируйте репозиторий:
  ```bash
   git clone [https://github.com/aibraikhan/network-device-scanner.git](https://github.com/aibraikhan/network-device-scanner.git)
  ```
3. Запустите скрипт:
  ```bash
  python network_device_scanner.py
  ```
